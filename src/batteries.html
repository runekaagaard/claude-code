Batteries

Task

Delegate complex work to specialized sub-agents.

Bash

Run shell commands: git, tests, builds.

Glob

Find files by pattern: **/*.ts, src/**/*.{js,jsx}.

Grep

Search file contents with regex and filters.

Read

View files with line numbers. Text, images, PDFs, notebooks.

Edit

Replace exact text matches. Preserves formatting.

Write

Create or overwrite entire files.

NotebookEdit

Modify Jupyter notebook cells.

WebFetch

Retrieve and analyze web pages.

WebSearch

Search for current information and recent docs.

TodoWrite

Track progress with task lists.

ExitPlanMode

Review implementation plans before coding.

BashOutput

Monitor output from background processes.

KillShell

Stop background processes.

SlashCommand

Run custom project commands from .claude/commands/.

gh (via Bash)

GitHub operations: gh pr create, gh issue list.
